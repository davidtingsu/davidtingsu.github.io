Tracking Cal Events

Integrate with Google Calendar

Using a calendaring system is tied with productivity hacking as discussed in James Maa’s productivity hacking. I first discovered that it is possible to embed calendars and my class schedule into my google calendar after using NinjaCourses. A public calendar in ics or ical format can be imported into Google calendar by following these steps. Take, for example my freshman fall 2010 schedule which can be viewed in html. For schedulebuilder users, because schedulebuilder uses virtually the same technology as ninja courses, you can import the events from link titled “Download ICal” on the “Saved Schedules” page.

Besides incorporating my class schedule into my calendar, I also import into my Google Calendar:

UC Berkeley Academic Calendar

The calendar url to embed into google calendar webcal://registrar.berkeley.edu/UCBAcadCal.ics and you can find other formats here. You will always know when school starts and when there is break, but make sure to take note of Add/Drop deadlines and P/NP Change of Grade Option deadlines which are not part of the calendar.

Department Events

I keep track of department events to find infosessions, undergraduate social hours, and seminars. After reading the documentation for exporting calendar events I was able to use http://events.berkeley.edu/index.php/ical/sn/eecs/type/month.htm as the embed url for EECS department sponsored events. To make sure that I also cover other non-department sponsored events, I also follow the facebook events of IEEE, Blueprint, CSUA, and Hackers@Berkeley. View a list of other campus calendars. For all of the calendar listed under UC Berkeley Calendar network, just reformat the url to be http://events.berkeley.edu/index.php/ical/sn/<deparment-name>/type/month.htm. I encourage you to dig around and search for calendar urls on the linked pages. For those of you particularly inclined in the visual and performing arts, consider Cal Performances and BAM/PFA.

Unsupported

Career Center events

You can find a list of Cal Career Center events (workshops and career centers) here, but the bottleneck is that you have download each of the events individually and there is currently know easy way to import the list of career center events into Google Calendar.
